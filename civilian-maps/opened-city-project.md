# 🏙️ Opened City Project

{% embed url="https://fivem.prompt-mods.com/package/5526512" %}

{% embed url="https://youtu.be/ac5wWgR_i1Q" %}

The **Opened City Project** is a custom map designed for FiveM servers, offering a fully immersive residential experience in the heart of Los Santos. This map features multiple open residential buildings with detailed interiors **(tp only)**, interactive props, and a robust doorlock system for enhanced gameplay. Below, you’ll find all the necessary details to install and configure this map on your server.



There were a lot fo questions on "How is this suppose to work?". The answer is each building has doors and 1 apartment interior in each building. The intention was to use those apartments as tp only. You need to teleport a user inside the same apartment but in different dimensions so they don't see each other. You can find more info about this [here](https://forum.cfx.re/t/free-standalone-configurable-world-panel-create-and-go-to-other-dimensions/4837293) .
