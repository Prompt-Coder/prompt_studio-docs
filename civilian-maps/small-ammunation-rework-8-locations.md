### 🚀 Installation Guide

Follow these steps to install the **Ammunation Rework** map on your FiveM server:

1. **Download the Map Resource**\
   Ensure the `prompt_ammunation_rework` resource is downloaded and placed in your `resources` folder.
2.  **Add the Resource to `server.cfg`**\
    Add the following line to your `server.cfg` to ensure the map loads when your server starts:

    ```
    start prompt_ammunation_rework
    ```
3. **Set Up Doorlocks**\
   Execute the provided SQL configuration in your database to enable the doorlock system.
4. **Test the Map**\
   Restart your server and visit one of the map locations to ensure everything is functioning correctly.
