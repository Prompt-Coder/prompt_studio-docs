# Elevator

A working multi-floor elevator connecting MRPD's levels.

***

### How it works

Interact with the elevator panel to pick a floor and travel between levels. Enable or disable the whole feature in `config.lua`:

```lua
Config.Modules.elevator = true
```

Floor definitions and the panel location live in the elevator module's config — adjust them there if your interior differs.
