# 🖥️ Scripts in Custom Maps

**🚀 Enhancing Your FiveM Experience with Scripts**

Some maps from **Prompt's Studio** come with built-in scripts to add interactive elements and functionality. These scripts bring your maps to life, offering features like working elevators, doors, or other interactive objects. For example, the **Sandy City Hall** map includes a fully functional elevator, adding a layer of realism and immersion to your server.

***

**🔧 How Do Scripts Work in Maps?**

1. **📦 Pre-Included Scripts**\
   All necessary scripts for a map are already included in the map’s resource folder. This means you don’t need to download or install additional scripts separately (at least for now). Simply install the map as instructed, and the scripts will work out of the box.
2. **📥 Easy Installation**\
   To use a map with scripts, follow the standard installation process:
   * Place the map resource in your `resources` folder.
   *   Add the map resource to your `server.cfg` file to ensure it loads when your server starts. For example:

       ```
       start sandycityhall
       ```
3. **⚙️ No Additional Configuration Needed**\
   The scripts are pre-configured to work seamlessly with the map. As long as the map resource is started correctly, the scripts will function automatically. Some scripts can have extra configuration files to suit your needs, but if there are such files, they are mentioned in the documentation.

***

**🌟 Explore Interactive Maps**

Maps with scripts offer a dynamic and engaging experience for your players. Whether it’s a working elevator, animated doors, or other interactive elements, these features are designed to enhance the realism and functionality of your FiveM server. Be sure to check the map’s documentation for details on what scripts are included and how they work.

***

✅ Ensure the map resource is placed in the `resources` folder.\
✅ Add the map resource to your `server.cfg`.\
✅ Start the server and enjoy the interactive features!\
✅ Refer to the map’s documentation for script details.
