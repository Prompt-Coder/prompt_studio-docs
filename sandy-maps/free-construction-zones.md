# 🏗️ \[FREE] Construction Zones

Two free standalone map packs that add construction-in-progress lots to Sandy Shores: one at the corner (bowling-alley + pub site) and one in the downtown area. Each pack is independent — install one, the other, or both. Includes fenced perimeters with gates, ground decals, props, bushes, and distant/LOD lights for night driving.

{% embed url="https://youtu.be/hNyPQzAY80o" %}

{% hint style="info" %}
Both packs are pure map resources — no framework, no scripts, no dependencies beyond the Sandy MapData prerequisite.
{% endhint %}

## Installation Instructions

For each pack you want to use, follow these steps:

{% stepper %}
{% step %}
Place the resource folder (`prompt_sandy_corner_construction` and/or `prompt_sandy_downtown_construction`) inside your `resources` directory.
{% endstep %}

{% step %}
Insert the following line(s) to ensure the map loads automatically: `start prompt_sandy_corner_construction` and/or `start prompt_sandy_downtown_construction`
{% endstep %}

{% step %}
Make sure you have the correct Sandy MapData installed.
{% endstep %}

{% step %}
Restart your server and visit the construction site location. If the lot, fences, props, and lights load correctly, installation was successful ✅
{% endstep %}
{% endstepper %}

## Sandy Corner Construction

A construction lot at the Sandy Shores corner — the future bowling-alley and pub site, fenced off and props laid out as if work is in progress.

**Includes:**

* Bowling-alley construction structure with ground decals
* Pub corner construction site
* Two custom construction gates (a / b)
* Surrounding bushes and props
* Distant lights and LOD lights for night

**Resource folder:** `prompt_sandy_corner_construction`

## Sandy Downtown Construction

A construction lot in downtown Sandy Shores — fenced perimeter with construction billboard, ground decals, prop sets, and night lighting.

**Includes:**

* Downtown construction ground with decals and detail textures
* Construction billboard prop
* Two custom construction gates (left / right)
* Two prop sets and surrounding bushes
* Distant lights and LOD lights for night

**Resource folder:** `prompt_sandy_downtown_construction`
